# Aeroknow — UAV Systems Educational Platform

[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-000000?style=for-the-badge&logo=vercel)](https://aeroknow.vercel.app)

Aeroknow is an advanced educational platform dedicated to autonomous Unmanned Aerial Vehicle (UAV) systems. It provides PhD-level curriculum covering guidance, navigation, control, sensor fusion, machine learning, and swarm intelligence.

The platform combines rigorous theoretical foundations with practical implementation through annotated Python code and interactive 3D simulations.

Live Demo: https://aeroknow.vercel.app

## About the Project

Aeroknow delivers in-depth, book-quality content on UAV autonomy and systems engineering. The curriculum places strong emphasis on GNSS-denied navigation, multi-sensor fusion, deep learning for perception, and multi-agent systems — topics of critical importance in modern military and civilian UAV applications.

Approximately 40% of the content focuses on artificial intelligence and machine learning integration with UAV systems.

## Key Features

- 10 comprehensive modules spanning fundamentals to advanced swarm intelligence
- Mathematically rigorous explanations with properly rendered equations
- Ready-to-run Python simulations using NumPy, SciPy, and PyTorch
- Interactive 3D visualizations to illustrate complex concepts
- Reflection checkpoints to reinforce learning
- No hardware required — fully simulation-based

## Curriculum Modules

1. UAV Systems Overview — Taxonomy, ISR architectures, and subsystem integration  
2. Flight Dynamics — 6-DoF mechanics, aerodynamics, and stability analysis  
3. GNC Fundamentals — PID, LQR, guidance laws, and control loops  
4. GNSS-Denied Navigation — EKF/UKF, Visual-Inertial Odometry, and terrain-aided navigation  
5. Sensor Fusion — IMU, LiDAR, camera fusion using Bayesian methods  
6. ML Foundations for UAVs — Supervised learning, Reinforcement Learning, and sim-to-real transfer  
7. Deep Learning and Perception — CNNs, Vision Transformers, object detection  
8. Autonomy and Decision-Making — POMDPs, behavior trees, and ethical considerations  
9. Swarm Systems — Multi-agent coordination and Multi-Agent Reinforcement Learning  
10. Systems Engineering and Reliability — MBSE, FMEA, airworthiness standards

**Topic Distribution**  
Machine Learning & Deep Learning (40%) • GNSS-Denied Navigation (20%) • Systems Engineering (20%) • GNC & Flight Dynamics (20%)

## Prerequisites

Recommended background:
- Mathematics: Linear algebra, calculus, probability and statistics, differential equations
- Programming: Intermediate Python with experience in NumPy, SciPy, and basic PyTorch
- Physics: Classical mechanics and basic aerodynamics

## Getting Started

1. Visit the live platform at https://aeroknow.vercel.app
2. Begin with the module "UAV Systems Overview"
3. Explore the embedded Python simulations and 3D visualizations

All content is designed to be self-contained and accessible without specialized hardware.

## Tech Stack

- Frontend: Next.js / React (deployed on Vercel)
- Mathematical rendering: MathJax
- Simulations: Python (NumPy, SciPy, PyTorch)
- 3D visualizations: Interactive web-based engine
