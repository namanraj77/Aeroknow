Curriculum

10 Book-Level Modules

Each module contains rigorous theory, MathJax-rendered equations, annotated Python simulations, and open-ended reflection checkpoints.

MODULE 01
UAV Systems Overview
Taxonomy, military mission profiles, ISR architectures, threat environments, and key subsystems.
ISR Strike Swarm
MODULE 02
Flight Dynamics
6-DOF rigid body mechanics, aerodynamics, rotorcraft vs fixed-wing, stability analysis, and Python simulation.
6-DOF Euler Angles
MODULE 03
GNC Fundamentals
PID, LQR, waypoint guidance, attitude controllers, inner/outer loops, and hardware-in-the-loop concepts.
PID LQR Guidance
MODULE 04
GNSS-Denied Navigation
EKF/UKF theory, visual-inertial odometry (VIO), terrain-aided navigation, and RF-contested environments.
EKF VIO TAN
MODULE 05
Sensor Fusion
IMU, camera, LiDAR, radar integration. Bayesian fusion, covariance modeling, temporal alignment.
Bayesian LiDAR IMU
MODULE 06
ML Foundations for UAV
Supervised/RL frameworks, optimization theory, dataset pipelines for UAV perception, sim-to-real gap.
Supervised RL Sim-to-Real
MODULE 07
Deep Learning & Perception
CNNs, ViTs, object detection (YOLO/DETR), optical flow, depth estimation, and UAV-specific datasets.
CNN ViT YOLO
MODULE 08
Autonomy & Decision-Making
POMDP, behavior trees, mission planning, threat response, OODA loop integration, lethal autonomy ethics.
POMDP BT Ethics
MODULE 09
Swarm Systems
Multi-agent coordination, consensus algorithms, mesh networking, anti-swarm countermeasures, MARL.
MARL Consensus Mesh
MODULE 10
Systems Engineering & Reliability
MBSE, FMEA, fault tolerance, DO-178C, MIL-STD-882, safety cases, and airworthiness for autonomy.
FMEA MIL-STD DO-178C
